# mabanana.github.io

## Resume

The resume is maintained as a single Markdown source file and automatically compiled into two PDF variants by GitHub Actions.

| File | Description |
|------|-------------|
| [`resume/resume.md`](resume/resume.md) | **Edit this** — the canonical resume source |
| [`Assets/Resume-ATS.pdf`](Assets/Resume-ATS.pdf) | ATS-optimised plain PDF (auto-generated) |
| [`Assets/Resume-Designed.pdf`](Assets/Resume-Designed.pdf) | Visually designed PDF (auto-generated) |

See [`resume/README.md`](resume/README.md) for full editing instructions and how to build locally.
